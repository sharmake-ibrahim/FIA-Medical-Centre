## Overview

This repository contains the design and implementation of a **relational SQL database** for **FIA Medical Centre**. The project focuses on modeling real-world healthcare operations using structured database design and SQL best practices.

## Description

The database supports core medical centre workflows such as managing **patients**, **staff**, **staff specialties**, **appointments**, and **consultations**. The schema is fully normalized and enforces data integrity through primary and foreign key relationships.

## Key Features

* Well-structured relational database schema
* Normalized tables to minimize data redundancy
* Clear entity relationships reflecting real medical workflows
* SQL scripts for database creation and management

## Technologies

* SQL
* Database Modeling
* ER Diagrams
* Normalization (1NF, 2NF, 3NF)

## Impact

* Improves accuracy and consistency of medical records
* Enables efficient retrieval of patient and consultation data
* Mirrors real-world healthcare database systems
* Provides a scalable foundation for future system expansion
